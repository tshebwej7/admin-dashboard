# Admin Dashboard

A professional responsive admin dashboard built with HTML, CSS, JavaScript, and Chart.js.

## Features

- Responsive Design
- Dark Mode
- Chart.js Analytics
- Sidebar Navigation
- Statistics Cards
- Recent Orders Table
- Local Storage Theme Persistence

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js
- Git
- GitHub
